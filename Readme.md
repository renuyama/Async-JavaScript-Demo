# Canine Carousel: Async JavaScript Demo

Canine Carousel is a demonstration project built to practice JavaScript
fetch calls, async/await, JSON parsing, and DOM manipulation. It retrieves
live dog breed data from a public API and generates an automated slideshow
based on the selected breed.
